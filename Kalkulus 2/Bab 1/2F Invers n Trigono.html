BAB 1 : FUNGSI TRANSENDEN


1.2 Fungsi Invers Trigonometri


A. Rangkuman Materi


1.1 Definisi Fungsi Hiperbolik

Definisi 1.1.1
Fungsi sinus hiperbolik dan cosinus hiperbolik, masing-masing dinotasikan dengan \(sinh\) dan \(cosh\), dan didefinisikan dengan

\( \sinh x= \frac{e^x-e^{-x}}{2} \)   \(\cosh x = \frac{e^x+e^{-x}}{2}\)


1.2 Kesamaan Hiperbolik

\(\cosh (-x)=\cosh x\)

\(\sinh (-x)=-\cosh x\)

\(\cosh^2 x - \sinh^2 x =1 \)

\(\tanh^2 x + sech^2 x =1 \)

\(\coth^2 x - csch^2 x =1 \)

\(\cosh x + \sinh x =e^x \)

\(\cosh x - \sinh x =e^{-x} \)

\(2\sinh x \cosh x =\sinh 2x \)

\(\cosh^2 x + \sinh^2 x =\cosh 2x \)

\(\sinh^2 x + 1 =\cosh 2x \)

\(\cosh^2 x - 1 =\cosh 2x \)

\(\sinh (x+y)=\sinh x \cosh y + \cosh x \sinh y\)

\(\sinh(x-y)=\sinh x \cosh y - \cosh x \sinh y\)

\(\cosh (x+y)=\cosh x \cosh y + \sinh x \sinh y\)

\(\cosh(x-y)=\cosh x \cosh y - \sinh x \sinh y\)

1.3 Rumus Turunan Fungsi Hiperbolik

    \(\frac{d}{dx}(\sinh u) = \cosh u \frac{du}{dx}\)

    \(\frac{d}{dx}(\cosh u) = \sinh u \frac{du}{dx}\)

    \(\frac{d}{dx}(\tanh u) = sech^2 u \frac{du}{dx}\)

    \(\frac{d}{dx}(\coth u) = -csch^2 u \frac{du}{dx}\)

    \(\frac{d}{dx}(sech u) = -sech u \tanh u \frac{du}{dx}\)

    \(\frac{d}{dx}(csch u) = -csch u \coth u \frac{du}{dx}\)

    1.4 Rumus Integral Fungsi Hiperbolik


    \(\int \sinh u \, du = \cosh u + C\)
    \(\int \cosh u \, du = \sinh u + C\)
    \(\int \tanh u \, du = \ln |\cosh u| + C\)
    \(\int \coth u \, du = \ln |\sinh u| + C\)
    \(\int sech^2 u \, du = \tanh u + C\)
    \(\int csch^2 u \, du = -\coth u + C\)
    \(\int sech u \tanh u \, du = -sech u + C\)
    \(\int csch u \coth u \, du = -csch u + C\)
    \)

    1.5 Invers Fungsi Hiperbolik

    Fungsi Domain Range Hubungan Dasar
    \(\sinh^{-1} x\) \((-\infty, +\infty)\) \((-\infty, +\infty)\) \(\sinh^{-1}(\sinh x) = x \text{ jika } -\infty < x < +\infty\)
    \(\sinh(\sinh^{-1} x) = x \text{ jika } -\infty < x < +\infty\)
    \(\cosh^{-1} x\) \([1, +\infty)\) \([0, +\infty)\) \(\cosh^{-1}(\cosh x) = x \text{ jika } x \ge 0\)
    \(\cosh(\cosh^{-1} x) = x \text{ jika } x \ge 1\)
    \(\tanh^{-1} x\) \((-1, 1)\) \((-\infty, +\infty)\) \(\tanh^{-1}(\tanh x) = x \text{ jika } -\infty < x < +\infty\)
    \(\tanh(\tanh^{-1} x) = x \text{ jika } -1 < x < 1\)
    \(\coth^{-1} x\) \((-\infty, -1) \cup (1, +\infty)\) \((-\infty, 0) \cup (0, +\infty)\) \(\coth^{-1}(\coth x) = x \text{ jika } x < 0 \text{ atau } x > 0\)
    \(\coth(\coth^{-1} x) = x \text{ jika } x < -1 \text{ atau } x > 1\)
    \(sech^{-1} x\) \((0, 1]\) \([0, +\infty)\) \(sech^{-1}(sech x) = x \text{ jika } x \ge 0\)
    \(sech(sech^{-1} x) = x \text{ jika } 0 < x \le 1\)
    \(csch^{-1} x\) \((-\infty, 0) \cup (0, +\infty)\) \((-\infty, 0) \cup (0, +\infty)\) \(csch^{-1}(csch x) = x \text{ jika } x < 0 \text{ atau } x > 0\)
    \(csch(csch^{-1} x) = x \text{ jika } x < 0 \text{ atau } x > 0\)
    Teorema 1.5.1

    \(y=\sinh^{-1}x \) ekuivalen dengan \(\sinh y = x\) untuk semua \(x,y\)

    \(y=\cosh^{-1}x \) ekuivalen dengan \(\cosh y = x\) jika \(\begin{cases} x \ge 1 \\ y \ge 0. \end{cases}\)

    \(y=\tanh^{-1}x \) ekuivalen dengan \(\tanh y = x\) jika \(\begin{cases} -1 < x < 1 \\ -\infty < y < +\infty. \end{cases}\)

    \(y=\coth^{-1}x \) ekuivalen dengan \(\coth y = x\) jika \(\begin{cases} |x| > 1 \\ y ≠ 0\. \end{cases}\)

    \(y=sech^{-1}x \) ekuivalen dengan \(sech y = x\) jika \(\begin{cases} 0 < x \le 1 \\ y \ge 0. \end{cases}\)

    \(y=\csch^{-1}x \) ekuivalen dengan \(csch y = x\) jika \(\begin{cases} x ≠ 0 \\ y ≠ 0. \end{cases}\)

    1.6 Bentuk Logaritma Fungsi Invers Hiperbolik

    Teorema 1.6.1
    Hubungan berikut ini berlaku, dengan pembatasan pada \(x\) dalam (1.5.1)

    \( \sinh x= \frac{e^x-e^{-x}}{2} \)   \(\cosh x = \frac{e^x+e^{-x}}{2}\)

1. Soal Kuis
Dapatkan nilai \(x\) dari: \(\ln (x+6) + \ln (x-3) = \ln 5 +\ln2\)

Pembahasan

ingat bahwa \(\ln (ab) = \ln a + \ln b\), maka

\(\ln (x+6) +\ln (x-3) = \ln 5 + \ln2\)

\(\ln (x+6)(x-3) = \ln 10\)

sehingga \((x+6)(x-3)=10 -> x^2 +3x -28 = (x+7)(x-4)=0\). Jadi x yang memenuhi adalah \(7\) atau \(4\)
2. Soal ETS 2020
Dapatkan turunan dari \(y=(1+x)^{\frac{1}{x}}\)

Pembahasan

\(y=(1+x)^{\frac{1}{x}}\)

\(\ln y =\ln(1+x)^{\frac{1}{x}}\)

\(\ln y=\frac{1}{x} \ln (1+x)\)

\(\frac{d}{dx}[\ln y]=\frac{1}{y}[\frac{d}{dx} \ln(1+x)]\)

\(\frac{dy}{dx}\frac{1}{y}=-\frac{1}{x^2} \ln(1+x)+\frac{1}{1+x}\frac{1}{x}\)

\(\frac{dy}{dx}=(-\frac{1}{x^2} \ln (1+x)+\frac{1}{1+x}\frac{1}{x})\)

\(\frac{dy}{dx}=(\frac{-(1+x) \ln (1+x)+x}{x^2(1+x)})(1+x)^{\frac{1}{x}}\)

\(\frac{dy}{dx}=(-\frac{-(1+x) \ln (1+x)+x}{x^2})(1+x)^{\frac{1-x}{x}}\)

3. Hitung \(\int \frac{dx}{x \ln x}\)
Pembahasan:
Misalkan \(u=\ln x\), sehingga \(du = \frac{dx}{x}\)

\(\int \frac{dx}{x\ln x}=\int \frac{du}{u}\)

=\(\ln x\) + C

=\(\ln (\ln x)\) + C

4. Soal ETS 2024 Hitung integral \(\int x2^{x^2};dx \)
Hint 1: Substitusi \(u=x^2\)
Hint 2: Gunakan sifat \(\int a^x; dx = \frac{a^x}{\ln a}+ C\)
Pembahasan:
Misalkan \(u=x^2\), sehingga \(du=2x dx\) ⇔ \( \frac{du}{2} = xdx\)

\(\ \int x2^{x^2} dx = \int 2^u\frac{du}{2} \)

\(=\ \frac{1}{2} \int 2^u ;du\)

\(= \frac{1}{2}\frac{2^u}{\ln 2} +C\)

\(= \frac{2^{u-1}}{\ln 2} + C\)

\(= \frac{2^{x^2-1}}{\ln 2} + C\)
